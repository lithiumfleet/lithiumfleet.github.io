1. 重积分计算方法

  • 投影: 分别对笛卡尔坐标的平面和轴投影. 然后使用.
  • 换系: 换成参数坐标系, 极坐标系, 柱坐标系, 球坐标系等. 目的是让变量间尽可能解耦, 方便投影.
  • 对称化简: 包括轮换对称.

2. 曲线曲面积分计算方法

  • 换系并改积分元: 如将笛卡尔坐标系中的曲面积分元改到柱坐标系下. 接着使用重积分计算方法即可

以上. 理论上来说多重积分可以使用上述两种(其实是一种)方法–计算. 下面介绍一些理论.

3. 积分关系定理

3.1 前置常识

3.1.1场的基本属性
  1. 场: 将空间中的每一个点都映射到一个标量/向量的函数.
  2. 标量场基本属性: 定义域 / 值域 / 连续性 / 可微性 / 等值面 / 梯度 / ……
  3. 向量场基本属性: 定义域 / 值域 / 连续性 / 可微性 / 等矢量面 / 旋度 / 散度 / ……
3.1.2 grad / rot / div
  1. 梯度(Grad): 是标量场的属性, 也可以理解为一个操作: 梯度操作是一种映射, 作用于定义在某个子空间中的标量场, 以该标量场的定义域为输出向量场的定义域, 将该标量场中的每个点映射为向量, 输出向量场. 向量为该点梯度值, 方向为最速增加方向.
  2. 散度(Div): 描述向量场某处的散敛性, 正值发散, 负值收敛.
  3. 旋度(Rot/Curl): 描述向量场某处旋转性质, 方向为旋转轴, 大小为旋转速度. 此处速度理解为”当空间位置改变时，场方向“扭曲”的速率，即矢量场旋转的趋势“[^1]

3.2 格林公式 / 高斯公式 / 斯托克斯公式

  1. 使用前: 保证闭合区域且内部可微, 选定正方向.
  2. 公式表示: ( 表示边界)

    点乘拆开后就是熟悉的 或 等
  3. 物理意义:
    • 格林公式: 通过边界流体速度计算区域内部的总涡度(流体旋转的强度).
    • 高斯公式: 通过向量场的散度计算某个闭合曲面的流量.
    • 用斯托克斯公式: 闭合线圈围成的曲面内部的磁场随时间变化率等于磁场旋度的负值, 其积分等于闭合线圈电场对路径积分, 那么接入电路的部分就可以通过积分算出电势.
